项目以及构建设计来源自[hoshino](https://github.com/Ice-Cirno/HoshinoBot)

基于Wechaty, 还未实现大部分功能, 仅供自我学习

### 首先安装依赖
```bash
pip3 install -r requirement.txt
```

### 启动程序
1. docker运行, 并登陆微信
```shell
bash wechaty.sh
```

2. 启动项目

    **linux**
    ```shell
    python3 run.py
    ```
    **windows**
    ```shell
    py run.py
    ```
